• HyperText Markup Language, usually abbreviated as HTML or, in its latest version, HTML5, is the markup language designed to represent web pages.

  • Cascading Style Sheets, commonly referred to as Cascading Style Sheets, are a computer language that describes the layout of HTML and XML documents. The standards defining CSS are published by the World Wide Web Consortium.

  • JavaScript is a scripting programming language primarily used in interactive web pages and as such is an essential part of web applications. Along with HTML and CSS, JavaScript is at the heart of the languages used by web developers.

  • React is a free JavaScript library. It has been maintained by Meta as well as a community of individual developers and companies since 2013.

  • Node.js is a free software platform in JavaScript, oriented towards highly competitive event-driven network applications that must be able to scale up.
